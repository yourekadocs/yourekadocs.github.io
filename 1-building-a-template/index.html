Building A Template

New Template

In order to create a new Template, click on the Form Templates tab and click the New button. When you click on the New button, it will prompt you to create a Form Template Name in the Designer page.

Designer

The Designer page allows you to create the Template Name and add Sections and Questions. You can also add various questions types (Text, Number, Picklists, Field Links, etc.) and create dependent questions.

Form Template name

Create a new Template by clicking on the New button on Form Templates page.

Clicking on the New button will prompt you to set a Template Name. The Template Name field will default to “New Template”. Change the name and click the OK button. !

You can edit the Template Name by clicking on the Edit icon next to Form Template name.

Helper Text

Helper Text on Form Template Designer page serves as a guideline to design a Template. It appears below the progress bar. It includes step by step instructions on how to build a Template. All green text in the Helper Text highlights the appropriate area on the designer. When a user clicks the X on the Helper Text, an alert modal appears to confirm if the Helper Text should be hidden permanently for that user freeing up the space for the designer.

Sections

After creating a Form Template, you can start adding sections by dragging the Section button down into the workspace.

The Section dialog will prompt you for a Section Name (required field) and for the Number of Columns (1 or 2). A page break can be added by selecting the checkbox. Click the Save button to create the new Section.

Linked Sections (BETA)

Linked Sections can be used to store repeating information. For example, if you are creating an inventory form, you can create a regular Section to record the date and person doing the count, and a Linked Section to hold the specific item and number in stock. Linked Sections can only be created for Objects that look up to the Form Object. In the example above, you need to create an Object (for example, Item_Count__c) with fields for the item, the count, and a lookup relationship field to the Form Object. Only Field Link Questions (see below) can be used in a Linked Section. All Field Link Questions in a Linked Section must refer to the same Object. To create a Linked Section, drag the Linked Section Button down to the workspace.

When creating a Linked Section, the Linked Section Object and its relationship field to the Form Object are required.

Note that: * Linked Sections is not yet supported in Youreka Mobile. * There are also additional considerations for Form Template Import/Export, as described in the Form Import section of this document. When importing a Template with Linked Section, both the linked Object and the relationship field to the Form__c Object must exist prior to the import.
